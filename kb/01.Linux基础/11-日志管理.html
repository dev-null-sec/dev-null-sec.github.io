еӨ§еӨҡж•°зі»з»ҹе’ҢжңҚеҠЎзҡ„ж—Ҙеҝ—йӣҶдёӯеңЁ /var/log дёӢгҖӮ

ж–Үд»¶еҜ№еә”еҸ‘иЎҢзүҲеҶ…е®№
/var/log/messagesRHEL зі»йҖҡз”Ёзі»з»ҹж—Ҙеҝ—
/var/log/syslogDebian/UbuntuйҖҡз”Ёзі»з»ҹж—Ҙеҝ—
/var/log/secureRHEL зі»и®ӨиҜҒе’Ңе®үе…Ёзӣёе…іж—Ҙеҝ—пјҲSSH зҷ»еҪ•гҖҒsudo зӯүпјү

дёҚеҗҢзҡ„еҸ‘иЎҢзүҲж—Ҙеҝ—ж–Үд»¶еҗҚдёҚе®Ңе…ЁзӣёеҗҢпјҢжҺ’жҹҘж—¶е…ҲзЎ®и®Өзӣ®ж ҮеҸ‘иЎҢзүҲзҡ„е‘ҪеҗҚд№ жғҜгҖӮйҒҮеҲ°зҷ»еҪ•зӣёе…ій—®йўҳзңӢ secure жҲ– auth.logпјҢеҶ…ж ёе’ҢзЎ¬д»¶й—®йўҳзңӢ dmesgгҖӮ

дәҢгҖҒjournald е’Ң journalctl вҖӢ

systemd зҡ„ж—Ҙеҝ—зі»з»ҹеҸ« journaldгҖӮе®ғж”¶жӢўдәҶ systemd з®ЎзҗҶжңҚеҠЎзҡ„ stdout/stderrгҖҒеҶ…ж ёж—Ҙеҝ—е’ҢйғЁеҲҶзі»з»ҹж—Ҙеҝ—пјҢеӯҳж”ҫеңЁдәҢиҝӣеҲ¶ж јејҸзҡ„ж—Ҙеҝ—ж–Үд»¶дёӯпјҲиҖҢдёҚжҳҜзәҜж–Үжң¬пјүпјҢжҹҘиҜўж—¶дҪҝз”Ё journalctlгҖӮ

bash
journalctl                        # жҹҘзңӢе…ЁйғЁж—Ҙеҝ—пјҲеҸҜиғҪйқһеёёеӨҡпјү
journalctl -u nginx               # еҸӘзңӢжҢҮе®ҡ unit зҡ„ж—Ҙеҝ—
journalctl -u nginx -f            # жҢҒз»ӯи·ҹиёӘпјҢзұ»дјј tail -f
journalctl -u nginx --since "1 hour ago"
journalctl -p err                 # еҸӘзңӢ error зә§еҲ«еҸҠд»ҘдёҠзҡ„ж—Ҙеҝ—

еёёз”ЁеҸӮж•°пјҡ

еҸӮж•°дҪңз”Ё
-uиҝҮж»ӨжҢҮе®ҡ systemd unit
-fжҢҒз»ӯиҫ“еҮәж–°ж—Ҙеҝ—пјҲfollowпјү
--sinceиө·е§Ӣж—¶й—ҙпјҲж”ҜжҢҒ "today"гҖҒ"1 hour ago"гҖҒ"2026-05-21"пјү
--untilз»“жқҹж—¶й—ҙ
-pжҢүж—Ҙеҝ—зә§еҲ«иҝҮж»ӨпјҲemergгҖҒalertгҖҒcritгҖҒerrгҖҒwarningгҖҒnoticeгҖҒinfoгҖҒdebugпјү
-nжҳҫзӨәжңҖиҝ‘ N иЎҢ
--no-pagerдёҚдҪҝз”ЁеҲҶйЎөзӣҙжҺҘиҫ“еҮәпјҢж–№дҫҝи„ҡжң¬еӨ„зҗҶ

жңҚеҠЎеҗҜеҠЁеӨұиҙҘзҡ„ж ҮеҮҶжҺ’жҹҘпјҡ

bash
systemctl status app --no-pager           # зңӢзҠ¶жҖҒе’ҢжңҖеҗҺеҮ иЎҢж—Ҙеҝ—
journalctl -u app -n 100 --no-pager       # зңӢжңҖиҝ‘ 100 иЎҢ

д»ҺжңҖиҝ‘ж—Ҙеҝ—ејҖе§ӢзңӢпјҢеҶҚжҢүж—¶й—ҙйҖҗжёҗжү©еӨ§иҢғеӣҙгҖӮдёҖдёҠжқҘе°ұзҝ»е…ЁйҮҸж—Ҙеҝ—ж•ҲзҺҮеҫҲдҪҺгҖӮ

journald зҡ„жҢҒд№…еҢ–иЎҢдёәдёҺ /var/log/journal зӣ®еҪ•жҳҜеҗҰеӯҳеңЁжңүе…ігҖӮеҰӮжһңиҜҘзӣ®еҪ•дёҚеӯҳеңЁпјҢjournald й»ҳи®ӨеҸӘеңЁ /run/log/journalпјҲtmpfsпјҢеҶ…еӯҳдёӯпјүеӯҳеӮЁж—Ҙеҝ—пјҢйҮҚеҗҜеҗҺж—Ҙеҝ—дёўеӨұгҖӮйңҖиҰҒжҢҒд№…еҢ–ж—¶пјҡ

bash
mkdir -p /var/log/journal
systemctl restart systemd-journald

дёүгҖҒrsyslog вҖӢ

rsyslog жҳҜдј з»ҹзҡ„зі»з»ҹж—Ҙеҝ—жңҚеҠЎпјҢиҙҹиҙЈжҢүи§„еҲҷе°ҶдёҚеҗҢжқҘжәҗзҡ„ж—Ҙеҝ—еҶҷеҲ°еҜ№еә”зҡ„ж–Үжң¬ж–Үд»¶дёӯгҖӮjournald е’Ң rsyslog еҸҜд»ҘеҗҢж—¶еӯҳеңЁвҖ”вҖ”journald ж”¶йӣҶпјҢеҶҚйҖҡиҝҮиҪ¬еҸ‘дәӨз»ҷ rsyslog еҪ’жЎЈгҖӮ

дё»й…ҚзҪ®е’Ңйҷ„еҠ й…ҚзҪ®пјҡ

rsyslog зҡ„и§„еҲҷж јејҸжҳҜ facility.level зӣ®ж Үпјҡ

conf
*.info;mail.none;authpriv.none;cron.none    /var/log/messages
authpriv.*                                  /var/log/secure
cron.*                                      /var/log/cron

facility иЎЁзӨәж—Ҙеҝ—жқҘжәҗзұ»еҲ«пјҲauthpriv=и®ӨиҜҒгҖҒcron=е®ҡж—¶д»»еҠЎгҖҒmail=йӮ®д»¶гҖҒdaemon=е®ҲжҠӨиҝӣзЁӢпјүпјҢlevel иЎЁзӨәи®°еҪ•зҡ„жңҖдҪҺзӯүзә§гҖӮauthpriv.* иЎЁзӨәи®ӨиҜҒзӣёе…ізҡ„жүҖжңүзә§еҲ«зҡ„ж—Ҙеҝ—йғҪеҶҷе…Ҙ /var/log/secureгҖӮ

дёҖиҲ¬зҡ„еә”з”ЁжңҚеҠЎдёҚзӣҙжҺҘдҝ®ж”№ rsyslog й…ҚзҪ®гҖӮеҸӘжңүеҪ“йңҖиҰҒеҒҡйӣҶдёӯж—Ҙеҝ—ж”¶йӣҶгҖҒиҮӘе®ҡд№үж—Ҙеҝ—и·Ҝеҫ„жҲ–и°ғж•ҙзі»з»ҹж—Ҙеҝ—иЎҢдёәж—¶жүҚдјҡеҠЁе®ғгҖӮ

еӣӣгҖҒdmesg еҶ…ж ёж—Ҙеҝ— вҖӢ

dmesg жҹҘзңӢеҶ…ж ёзҺҜеҪўзј“еҶІеҢәдёӯзҡ„ж¶ҲжҒҜпјҢеҢ…еҗ«зЎ¬д»¶жҺўжөӢгҖҒй©ұеҠЁеҠ иҪҪгҖҒOOM killerгҖҒзҪ‘з»ңжҺҘеҸЈзҠ¶жҖҒзӯүдҝЎжҒҜгҖӮ

зЈҒзӣҳ I/O й”ҷиҜҜгҖҒзҪ‘еҚЎ link downгҖҒOOM иў«жқҖиҝӣзЁӢгҖҒUSB и®ҫеӨҮејӮеёёзӯүйғҪеҸҜд»ҘеңЁ dmesg йҮҢжүҫеҲ°з—•иҝ№гҖӮйңҖиҰҒжіЁж„Ҹ dmesg -T жҳҫзӨәзҡ„ж—¶й—ҙеңЁжҹҗдәӣзі»з»ҹдёҠеҸҜиғҪдёҚеҮҶвҖ”вҖ”еҶ…ж ёзҡ„ж—¶й—ҙжҲіе’Ңзі»з»ҹж—¶й—ҙдёҚдёҖе®ҡеҗҢжӯҘпјҢе…ій”®дәӢж•…иҝҳжҳҜиҰҒз»“еҗҲ journald е’Ңе…¶д»–ж—Ҙеҝ—дәӨеҸүйӘҢиҜҒгҖӮ

дёҖдёӘе…ёеһӢзҡ„иҪ®иҪ¬й…ҚзҪ®пјҡ

logrotate
/var/log/myapp/*.log {
    daily                    # жҜҸеӨ©иҪ®иҪ¬дёҖж¬Ў
    rotate 7                 # дҝқз•ҷжңҖиҝ‘ 7 д»Ҫ
    missingok               # ж–Үд»¶дёҚеӯҳеңЁдёҚжҠҘй”ҷ
    notifempty              # з©әж–Үд»¶дёҚиҪ®иҪ¬
    compress                # еҺӢзј©ж—§ж—Ҙеҝ—
    delaycompress           # е»¶иҝҹеҺӢзј©пјҲдҝқз•ҷжңҖиҝ‘дёҖд»ҪдёҚеҺӢзј©пјҢж–№дҫҝд»ҚеңЁжҹҘпјү
    copytruncate            # жӢ·иҙқеҗҺжё…з©әеҺҹж–Үд»¶
}

copytruncate зҡ„ж–№ејҸпјҡе…ҲжҠҠж—Ҙеҝ—еҶ…е®№еӨҚеҲ¶еҲ°иҪ®иҪ¬ж–Үд»¶пјҢеҶҚжё…з©әеҺҹж—Ҙеҝ—ж–Үд»¶гҖӮдјҳзӮ№жҳҜдёҚйңҖиҰҒжңҚеҠЎж”ҜжҢҒйҮҚж–°жү“ејҖж—Ҙеҝ—ж–Үд»¶пјҢзјәзӮ№жҳҜеӨҚеҲ¶е’Ңжё…з©әд№Ӣй—ҙдә§з”ҹзҡ„е°‘йҮҸж—Ҙеҝ—еҸҜиғҪдёўеӨұгҖӮ

еҜ№дәҺж”ҜжҢҒ reload дҝЎеҸ·зҡ„жңҚеҠЎпјҲеҰӮ nginxпјүпјҢжӣҙеҸҜйқ зҡ„ж–№ејҸжҳҜ postrotateпјҡ

logrotate
/var/log/nginx/*.log {
    daily
    rotate 14
    missingok
    notifempty
    compress
    delaycompress
    postrotate
        systemctl reload nginx >/dev/null 2>&1 || true
    endscript
}

иҪ®иҪ¬е®ҢжҲҗеҗҺеҸ‘ reload дҝЎеҸ·з»ҷжңҚеҠЎпјҢи®©жңҚеҠЎйҮҚж–°жү“ејҖж—Ҙеҝ—ж–Үд»¶гҖӮиҝҷжҜ” copytruncate жӣҙеҸҜйқ пјҢдёҚдјҡдёўж—Ҙеҝ—гҖӮ

жөӢиҜ•й…ҚзҪ®пјҡ

bash
