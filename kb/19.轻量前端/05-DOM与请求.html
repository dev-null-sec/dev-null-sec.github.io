DOM与请求 ​

JavaScript 语言本身处理数据和逻辑。页面上的标题、段落、输入框、按钮属于 DOM。DOM 是浏览器把 HTML 解析后形成的页面对象树，JavaScript 可以读取它、修改它、监听事件。

请求部分先用本地 JSON 文件做示例，用来练习 fetch、Promise、async/await、loading、error、empty 状态。数据仍然来自静态文件，不处理保存和校验。

一、DOM对象 ​

HTML：

html
<h1 id="title">资料整理</h1>
<p class="note">第一条说明。</p>
<p class="note">第二条说明。</p>

JavaScript 读取元素：

js
const title = document.querySelector("#title");
const firstNote = document.querySelector(".note");
const notes = document.querySelectorAll(".note");

console.log(title.textContent);
console.log(firstNote.textContent);
console.log(notes.length);

常用方法：

方法用途
querySelector找到第一个匹配元素
querySelectorAll找到所有匹配元素
getElementById按 id 找元素

querySelector 使用 CSS 选择器写法，#title 表示 id，.note 表示 class。

二、修改内容 ​

修改文本：

html
<h1 id="title">旧标题</h1>
js
const title = document.querySelector("#title");
title.textContent = "新标题";

修改 HTML：

html
<div id="container"></div>
js
const container = document.querySelector("#container");
container.innerHTML = "<strong>重要内容</strong>";

textContent 按纯文本处理，innerHTML 会按 HTML 解析。显示外部输入时优先用 textContent，避免把字符串里的标签当成真实 HTML 执行。

修改属性：

html
<a id="link" href="about.html">关于</a>
js
const link = document.querySelector("#link");
link.href = "contact.html";
link.textContent = "联系";

修改 class：

js
const box = document.querySelector(".box");

box.classList.add("active");
box.classList.remove("hidden");
box.classList.toggle("selected");

classList 比直接拼接 className 更清楚，也不容易把原有 class 覆盖掉。

三、创建元素 ​

DOM 可以通过 JavaScript 动态创建。

html
<ul id="list"></ul>
js
const list = document.querySelector("#list");
const item = document.createElement("li");

item.textContent = "HTML 基础";
list.appendChild(item);

渲染数组：

js
const courses = ["HTML", "CSS", "JavaScript"];
const list = document.querySelector("#list");

list.innerHTML = "";

for (const course of courses) {
  const item = document.createElement("li");
  item.textContent = course;
  list.appendChild(item);
}

list.innerHTML = "" 用于清空旧内容。重新渲染列表前常见这一步。

也可以用字符串生成：

js
const courses = ["HTML", "CSS", "JavaScript"];
const list = document.querySelector("#list");

list.innerHTML = courses
  .map((course) => `<li>${course}</li>`)
  .join("");

字符串生成写起来短，但内容来自外部输入时要注意转义问题。基础阶段用 createElement 更容易看清每一步。

四、事件 ​

事件表示页面上发生了某个动作，比如点击按钮、输入文字、提交表单。

html
<button id="button" type="button">点击</button>
<p id="message"></p>
js
const button = document.querySelector("#button");
const message = document.querySelector("#message");

button.addEventListener("click", () => {
  message.textContent = "按钮被点击了";
});

常见事件：

事件触发时机
click点击元素
input输入框内容变化
change表单值变化
submit表单提交
keydown键盘按下

输入事件：

html
<input id="name" type="text">
<p id="preview"></p>
js
const input = document.querySelector("#name");
const preview = document.querySelector("#preview");

input.addEventListener("input", () => {
  preview.textContent = input.value;
});

input.value 读取输入框当前值。

五、表单处理 ​

表单默认提交会刷新页面。用 JavaScript 处理时，通常先阻止默认提交，再读取字段。

html
<form id="message-form">
  <label for="name">姓名</label>
  <input id="name" name="name" type="text" required>

  <label for="content">内容</label>
  <textarea id="content" name="content"></textarea>

  <button type="submit">提交</button>
</form>

<pre id="result"></pre>
js
const form = document.querySelector("#message-form");
const result = document.querySelector("#result");

form.addEventListener("submit", (event) => {
  event.preventDefault();

  const formData = new FormData(form);
  const data = Object.fromEntries(formData.entries());

  result.textContent = JSON.stringify(data, null, 2);
});

FormData 会按表单字段的 name 收集值。输入框缺少 name 时，提交数据里不会有这个字段。

简单校验：

js
if (!data.name) {
  result.textContent = "姓名不能为空";
  return;
}

浏览器自带的 required 可以做基础校验，JavaScript 校验适合补充更具体的规则。

六、fetch请求 ​

fetch 用来发起 HTTP 请求。基础阶段先读取本地 JSON 文件。

文件：data/courses.json

json
[
  { "title": "HTML", "status": "已完成" },
  { "title": "CSS", "status": "整理中" },
  { "title": "JavaScript", "status": "待开始" }
]

读取 JSON：

js
fetch("data/courses.json")
  .then((response) => response.json())
  .then((data) => {
    console.log(data);
  });

fetch 返回 Promise。response.json() 也返回 Promise，所以这里有两个 then。

async/await 写法：

js
async function loadCourses() {
  const response = await fetch("data/courses.json");
  const data = await response.json();

  console.log(data);
}

loadCourses();

await 只能放在 async 函数里。它让异步代码读起来接近同步顺序。

七、请求状态 ​

页面请求数据时常见三种状态：加载中、加载失败、没有数据。

html
<p id="status">加载中...</p>
<ul id="list"></ul>
js
const statusText = document.querySelector("#status");
const list = document.querySelector("#list");

async function loadCourses() {
  statusText.textContent = "加载中...";
  list.innerHTML = "";

  try {
    const response = await fetch("data/courses.json");

    if (!response.ok) {
      throw new Error(`请求失败：${response.status}`);
    }

    const courses = await response.json();

    if (courses.length === 0) {
      statusText.textContent = "暂无数据";
      return;
    }

    statusText.textContent = "";
    for (const course of courses) {
      const item = document.createElement("li");
      item.textContent = `${course.title} - ${course.status}`;
      list.appendChild(item);
    }
  } catch (error) {
    statusText.textContent = error.message;
  }
}

loadCourses();

response.ok 表示状态码在 200-299。文件不存在时，浏览器可能返回 404，这时要进入错误状态。

八、完整示例 ​

这个示例从本地 JSON 文件读取课程安排，渲染到列表里，并提供一个表单临时追加数据。新增的数据只存在浏览器内存里，刷新后会丢失。

文件：index.html

html
<!doctype html>
<html lang="zh-CN">
  <head>
    <meta charset="utf-8">
    <title>课程安排</title>
  </head>
  <body>
    <h1>课程安排</h1>

    <p id="status">加载中...</p>
    <ul id="course-list"></ul>

    <form id="course-form">
      <label for="title">主题</label>
      <input id="title" name="title" type="text" required>

      <label for="status-input">状态</label>
      <select id="status-input" name="status">
        <option value="待开始">待开始</option>
        <option value="整理中">整理中</option>
        <option value="已完成">已完成</option>
      </select>

      <button type="submit">添加</button>
    </form>

    <script src="app.js"></script>
  </body>
</html>

文件：data/courses.json

json
[
  { "title": "HTML", "status": "已完成" },
  { "title": "CSS", "status": "整理中" }
]

文件：app.js

js
const statusText = document.querySelector("#status");
const list = document.querySelector("#course-list");
const form = document.querySelector("#course-form");

let courses = [];

function renderCourses() {
  list.innerHTML = "";

  if (courses.length === 0) {
    statusText.textContent = "暂无数据";
    return;
  }

  statusText.textContent = "";
  for (const course of courses) {
    const item = document.createElement("li");
    item.textContent = `${course.title} - ${course.status}`;
    list.appendChild(item);
  }
}

async function loadCourses() {
  statusText.textContent = "加载中...";

  try {
    const response = await fetch("data/courses.json");
    if (!response.ok) {
      // response.ok 为 false 时，说明状态码不在 200-299。
      throw new Error(`请求失败：${response.status}`);
    }

    courses = await response.json();
    renderCourses();
  } catch (error) {
    statusText.textContent = error.message;
  }
}

form.addEventListener("submit", (event) => {
  event.preventDefault();

  const formData = new FormData(form);
  // 新增内容只放进当前数组，刷新页面后会重新读取 JSON。
  courses.push(Object.fromEntries(formData.entries()));
  form.reset();
  renderCourses();
});

loadCourses();

本地直接用文件路径打开页面时，部分浏览器会限制 fetch 读取本地 JSON。可以用一个静态文件服务打开目录，例如 VS Code Live Server，或者 Vite 开发服务。

九、常见问题 ​

现象常见原因排查入口
元素读不到JS 执行时 HTML 还没加载，或选择器写错Elements、Console
表单提交后刷新没有调用 event.preventDefault()submit 事件
字段没有进入 FormData输入框缺少 name表单 HTML
fetch 404路径写错或文件不存在Network
JSON 解析失败文件不是合法 JSONResponse 内容
页面没有显示错误catch 里没有更新状态文字异常处理

DOM 问题看 Elements 和 Console，请求问题看 Network。看到请求路径、状态码、响应内容后，很多问题会直接暴露出来。
