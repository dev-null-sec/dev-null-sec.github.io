01｜Python 是什么与安装 ​

Python 是一门解释型、动态类型、通用编程语言。它的语法接近自然语言，代码可读性高，标准库覆盖了文件处理、网络通信、文本解析、数据序列化等常见任务。

在自动化运维、数据分析、Web 开发、人工智能等领域，Python 都是主流选择之一。它的核心优势不是执行速度，而是开发效率和生态丰富度——同样的功能，用 Python 实现通常代码量更少，第三方库也更容易找到。

一、解释型语言 ​

Python 代码不是直接交给操作系统执行的。它需要先经过解释器（interpreter），把 .py 文件逐行翻译成机器能理解的指令。

这个流程意味着：

  • 运行 Python 程序前，目标机器上必须有对应版本的解释器
  • 同一段代码在不同操作系统上都能运行（跨平台）
  • 解释器版本很重要——Python 3.6 能跑的 f-string，Python 3.5 会直接报语法错误

二、安装 Python ​

Linux ​

RHEL / Rocky / CentOS：

bash
dnf install -y python3 python3-pip
python3 --version

Ubuntu / Debian：

bash
apt-get update
apt-get install -y python3 python3-venv python3-pip
python3 --version

系统仓库的 Python 版本可能偏旧。需要新版本时，从 python.org 下载源码或二进制包自行安装。

Windows ​

从 python.org 下载安装包，安装时勾选 "Add Python to PATH"。安装完成后：

powershell
py --version
py -0p

py 是 Windows 上的 Python 启动器，可以管理多个 Python 版本。py -0p 列出所有已安装的版本和路径。

macOS ​

bash
brew install python3
python3 --version

三、版本选择 ​

当前 Python 有两个大版本：

版本状态
Python 2已于 2020 年停止维护，不应在新项目中使用
Python 3当前主流，Python 3.9 及以上版本推荐

新学习的项目直接选 Python 3。Python 3.8 之后的版本引入了 := 海象运算符、更精确的类型注解、match-case 语法等特性。如果教程里用到这些语法，需要确认解释器版本是否支持。

四、验证安装 ​

安装完成后，确认三个基本要素：

bash
python3 --version      # 解释器版本
which python3          # 解释器路径
python3 -m pip --version  # pip 版本

python3 -m pip 是用当前解释器运行 pip 模块。当系统上同时存在 python、python3、py 多个命令时，这个写法能确保用的是同一个解释器对应的包管理器，避免装错环境。

记忆锚点：Python 是解释型语言，需要解释器才能运行；Python 2 已停止维护，新项目用 Python 3；python3 -m pip 确保包装到正确的环境里。
