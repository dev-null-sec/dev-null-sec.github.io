зЁӢеәҸиҝҗиЎҢж—¶йңҖиҰҒи®°еҪ•еҸ‘з”ҹдәҶд»Җд№ҲвҖ”вҖ”жҲҗеҠҹгҖҒеӨұиҙҘгҖҒејӮеёёгҖҒзҠ¶жҖҒеҸҳеҢ–гҖӮж—Ҙеҝ—жҳҜжҺ’жҹҘй—®йўҳзҡ„з¬¬дёҖжүӢжқҗж–ҷгҖӮй…ҚзҪ®еҲҷжҠҠдјҡеҸҳеҠЁзҡ„еҖјпјҲж–Үд»¶и·Ҝеҫ„гҖҒи¶…ж—¶ж—¶й—ҙгҖҒзӣ®ж Үең°еқҖпјүд»Һд»Јз ҒдёӯеҲҶзҰ»еҮәжқҘпјҢи®©и„ҡжң¬еңЁдёҚеҗҢзҺҜеўғдёӢеӨҚз”ЁгҖӮ

дёҖгҖҒlogging жЁЎеқ— вҖӢ

print() йҖӮеҗҲдёҖж¬ЎжҖ§и„ҡжң¬гҖӮзЁҚжӯЈејҸзҡ„зЁӢеәҸз”Ё loggingпјҢиғҪеҢәеҲҶзә§еҲ«гҖҒеёҰж—¶й—ҙжҲігҖҒиҝҳиғҪеҶҷж–Үд»¶гҖӮ

python
import logging

logging.basicConfig(
    level=logging.INFO,
    format="%(asctime)s %(levelname)s %(message)s",
)

logging.debug("и°ғиҜ•дҝЎжҒҜ")
logging.info("зЁӢеәҸеҗҜеҠЁ")
logging.warning("зЈҒзӣҳз©әй—ҙдёҚи¶і")
logging.error("иҝһжҺҘеӨұиҙҘ")
logging.critical("зі»з»ҹеҙ©жәғ")

ж—Ҙеҝ—зә§еҲ«пјҲд»ҺдҪҺеҲ°й«ҳпјүпјҡ

python
import logging

logger = logging.getLogger("myapp")
logger.setLevel(logging.DEBUG)

# ж–Үд»¶еӨ„зҗҶеҷЁ
file_handler = logging.FileHandler("app.log")
file_handler.setLevel(logging.INFO)

# жҺ§еҲ¶еҸ°еӨ„зҗҶеҷЁ
console_handler = logging.StreamHandler()
console_handler.setLevel(logging.DEBUG)

file_handler.setFormatter(formatter)
console_handler.setFormatter(formatter)

logger.addHandler(console_handler)

JSON жІЎжңүжіЁйҮҠпјҢдёҚйҖӮеҗҲйңҖиҰҒжіЁйҮҠиҜҙжҳҺзҡ„й…ҚзҪ®ж–Үд»¶гҖӮINI е’Ң YAML ж”ҜжҢҒжіЁйҮҠгҖӮ

е…«гҖҒй…ҚзҪ®йҖүжӢ© вҖӢ

ж јејҸдјҳзӮ№зјәзӮ№йҖӮз”ЁеңәжҷҜ
INIз®ҖеҚ•гҖҒжңүжіЁйҮҠгҖҒж ҮеҮҶеә“ж”ҜжҢҒдёҚж”ҜжҢҒеөҢеҘ—з®ҖеҚ•й”®еҖјй…ҚзҪ®
YAMLж”ҜжҢҒеөҢеҘ—гҖҒжңүжіЁйҮҠгҖҒеҸҜиҜ»жҖ§еҘҪйңҖиҰҒз¬¬дёүж–№еә“еӨҚжқӮй…ҚзҪ®
JSONж ҮеҮҶж јејҸгҖҒи§ЈжһҗеҷЁ ubiquitousж— жіЁйҮҠгҖҒдёҚж”ҜжҢҒеӨҡиЎҢеӯ—з¬ҰдёІзЁӢеәҸй—ҙдәӨжҚўй…ҚзҪ®
зҺҜеўғеҸҳйҮҸе®№еҷЁеҸӢеҘҪгҖҒдёҚиҗҪең°еҸӘж”ҜжҢҒеӯ—з¬ҰдёІж•Ҹж„ҹдҝЎжҒҜгҖҒе®№еҷЁйғЁзҪІ

и®°еҝҶй”ҡзӮ№пјҡlogging.basicConfig() еҝ«йҖҹй…ҚзҪ®ж—Ҙеҝ—пјӣж—Ҙеҝ—зә§еҲ« DEBUG < INFO < WARNING < ERROR < CRITICALпјӣз”Ё %s еҚ дҪҚз¬ҰиҖҢдёҚжҳҜ f-stringпјӣж–Үд»¶+жҺ§еҲ¶еҸ°еҗҢж—¶иҫ“еҮәз”ЁеӨҡдёӘ HandlerпјӣINI з”Ё configparserпјҢYAML з”Ё pyyamlпјҢJSON з”Ёж ҮеҮҶеә“пјӣyaml.safe_load() жҜ” load е®үе…Ёпјӣж•Ҹж„ҹдҝЎжҒҜж”ҫзҺҜеўғеҸҳйҮҸгҖӮ
